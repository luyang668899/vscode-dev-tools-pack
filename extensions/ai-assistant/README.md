# AI Assistant Extension

AI-powered coding assistant for Visual Studio Code, providing intelligent code generation, explanation, refactoring, and performance optimization capabilities.

## üöÄ Features

- **Code Generation**: Generate code from natural language prompts
- **Code Explanation**: Explain complex code in plain English
- **Code Refactoring**: Suggest and apply code refactoring
- **Comment Generation**: Automatically generate code comments
- **Refactoring Suggestions**: Get intelligent refactoring suggestions
- **Performance Optimization**: Optimize code performance with AI suggestions

## üì¶ Installation

1. Download the `ai-assistant-1.0.0.vsix` file
2. In VS Code, go to Extensions ‚Üí More Actions ‚Üí Install from VSIX...
3. Select the downloaded VSIX file
4. Reload VS Code to activate the extension

## üõ†Ô∏è Usage

### Command Palette

Open the Command Palette (`Ctrl+Shift+P`) and search for "AI Assistant" to see all available commands:

- **AI Assistant: Generate Code** - Generate code from a prompt
- **AI Assistant: Explain Code** - Explain selected code
- **AI Assistant: Refactor Code** - Refactor selected code
- **AI Assistant: Generate Comments** - Generate comments for code
- **AI Assistant: Suggest Refactoring** - Get refactoring suggestions
- **AI Assistant: Optimize Performance** - Optimize code performance

### Context Menu

Right-click in the editor to access AI Assistant commands from the context menu.

## üìù Examples

### Generating Code

1. Press `Ctrl+Shift+G` or run "AI Assistant: Generate Code" from the Command Palette
2. Enter a prompt like: "Create a TypeScript function that calculates Fibonacci numbers"
3. The generated code will be inserted into your editor

### Explaining Code

1. Select a piece of code in the editor
2. Press `Ctrl+Shift+E` or run "AI Assistant: Explain Code"
3. A webview will open with a detailed explanation of the code

### Refactoring Code

1. Select a piece of code that needs refactoring
2. Press `Ctrl+Shift+R` or run "AI Assistant: Refactor Code"
3. The code will be refactored with improved structure

## üîß Configuration

The AI Assistant extension doesn't require any specific configuration. It works out of the box with sensible defaults.

## üêõ Troubleshooting

### Common Issues

- **Command not found**: Make sure the extension is installed and VS Code is reloaded
- **No response**: Check that you have an active internet connection
- **Generated code quality**: The quality of generated code depends on the clarity of your prompt

### Known Limitations

- The extension uses simulated AI responses for demonstration purposes
- Performance optimization suggestions are based on common patterns
- Refactoring suggestions may not cover all possible improvements
